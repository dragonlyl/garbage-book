# git

## git stash

[Git stash](https://juejin.cn/post/7071780876501123085)
切换分支, 但是有内容没有提交 ,可以使用 git stash 暂存 (类似commit 但是不会有提交记录)
使用 git stash apply  从暂存的地方拿出来
gsta   git stash
gstaa  git stash apply
gstl  git stash list

## git reset --soft HEAD^  

commit 太多想回退

使用上述指令 回退最近一次commit

## git cherry-pick

挑选一个分支的部分commit 到另外一个分支上
