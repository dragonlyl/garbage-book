## 对象转化

先 valueOf  然后toString，同时你可以重写这两个方法
let a = {
    valueOf() {
        return 0
    }
}
这样 拿a去做数字运算a就会被转变成0进行运算
>当然你也可以重写 Symbol.toPrimitive ，该方法在转基本类型时调用优先级最高。

``` // 按优先级转化
let a = {
    valueOf() {
        return 0;
  },
  toString() {
    return '1';
  },
  [Symbol.toPrimitive]() {
    return 2;
  }
}
1 + a // => 3
'1' + a // => '12' // 把symbol那行去掉输出的是10 并不是11
```

### 四则运算

对于加法 其中有字符串类型，就全转字符串；其他运算，其中有数字就全转数字
1 + '1' // '11'
2 * '2' // 4
[1, 2] + [2, 1] // '1,22,1'
// [1, 2].toString() -> '1,2'
// [2, 1].toString() -> '2,1'
// '1,2' + '2,1' = '1,22,1'
