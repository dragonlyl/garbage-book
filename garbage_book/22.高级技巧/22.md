## 作用域安全的构造函数

如果用new 构造函数生成的时候忘记写new  就会导致this被绑定到window对象
所以 加个保证

``` // 保证
function Person (name) {
    if(this instanceof Person) {
        this.name = name;
    }else {
        return new Person (name);
    }
}
```
