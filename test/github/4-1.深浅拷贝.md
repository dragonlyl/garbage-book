# 深浅拷贝

* 基本数据类型：赋值，赋值之后两个变量互不影响
* 引用数据类型：赋址，两个变量具有相同的引用，指向同一个对象，相互之间有影响

1. JSON.parse(JSON.stringify(object))
    但是该方法有以下几个问题。
    1、会忽略 undefined

    2、会忽略 symbol

    3、不能序列化函数

    4、不能解决循环引用的对象

    5、不能正确处理new Date()

    6、不能处理正则

```JSON
    let obj = {
        name: 'muyiy',
        a: undefined,
        b: Symbol('muyiy'),
        c: function() {}
    }
    console.log(obj);
    // {
    //     name: "muyiy",
    //     a: undefined,
    //     b: Symbol(muyiy),
    //     c: ƒ ()
    // }

    let b = JSON.parse(JSON.stringify(obj));
    console.log(b);// {name: "muyiy"}
```
